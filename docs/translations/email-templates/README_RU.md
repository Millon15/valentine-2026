# Шаблон EmailJS

Этот каталог содержит HTML-шаблон письма и документацию по настройке для приложения викторины ко Дню святого Валентина.

## Файлы

- **`quiz-answers.html`** - Готовый к использованию HTML-шаблон письма
- **`SETUP.md`** - Полное руководство по настройке EmailJS (создание аккаунта, настройка сервиса, конфигурация шаблона)
- **`README.md`** - Этот файл (документация шаблона и использование)

## Шаблон: `quiz-answers.html`

Красиво оформленный шаблон письма, который отправляет результаты викторины с:

### Возможности

- **Градиентный заголовок** с украшениями в виде эмодзи 💕
- **Детали участника**: раздел с именем, email и временем отправки
- **Ответы викторины**: отображаются в форматированном блоке кода
- **Превью любовного письма** (необязательно, отображается условно)
- **Значок идеального результата** (всегда 100% ❤️)
- **Адаптивный дизайн**: оптимизирован для почтовых клиентов на десктопах и мобильных
- **Кнопка призыва к действию**: ссылка на живую викторину
- **Ссылка на GitHub Star** в футере

### Особенности дизайна

- Современные градиентные фоны (розовый, фиолетовый, коралловый)
- Закругленные углы и мягкие тени
- Чистая типографика с системными шрифтами
- Акценты с эмодзи повсюду
- Мобильная адаптивность (макс. ширина: 600px)
- Inline CSS для максимальной совместимости с почтовыми клиентами

### Переменные EmailJS

Шаблон использует следующие переменные EmailJS (заполняются автоматически):

| Переменная        | Описание                 | Пример                           |
|-------------------|--------------------------|----------------------------------|
| `{{user_name}}`   | Имя участника            | "Таня"                           |
| `{{user_email}}`  | Email участника          | "tanya@example.com"              |
| `{{timestamp}}`   | Время отправки           | "2026-02-14 18:30:00"            |
| `{{answers}}`     | Ответы (JSON/текст)      | Форматированная строка Q&A       |
| `{{love_letter}}` | Сгенерированное письмо   | Опционально, рендерится условно  |

### Использование в панели EmailJS

**Настраиваете впервые?** См. [SETUP.md](./SETUP.md) для пошаговых инструкций.

**Быстрая настройка шаблона**:

1. Перейдите в [EmailJS Templates](https://dashboard.emailjs.com/admin/templates)
2. Нажмите **"Create New Template"**
3. Скопируйте содержимое `quiz-answers.html`
4. Вставьте в редактор **HTML Content**
5. Установите **Тему письма (Subject Line)**: `💕 New Valentine's Quiz Submission from {{user_name}}`
6. Сохраните и скопируйте **Template ID**
7. Добавьте Template ID в ваш файл `.env`

### Кастомизация

Чтобы настроить шаблон:

- **Цвета**: Измените значения `linear-gradient()`
- **Шрифты**: Измените `font-family` в стилях body
- **Логотип**: Добавьте URL изображения или используйте эмодзи
- **Ссылки в футере**: Обновите ссылку на GitHub на ваш репозиторий
- **Призыв к действию**: Измените URL кнопки на ваш развернутый сайт

### Тестирование

Отправьте тестовое письмо из панели EmailJS:

1. Откройте шаблон в EmailJS
2. Нажмите **"Test It"**
3. Заполните примеры значений для всех переменных
4. Отправьте на свой email
5. Проверьте входящие (и папку спам)
